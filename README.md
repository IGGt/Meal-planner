# Gates Family Meal Planner

A simple, elegant web-based meal planning application designed for families to organize their weekly meals with ease.

## What It Does
The Gates Family Meal Planner helps you plan meals for each day of the week across multiple weeks. It provides an intuitive interface to track breakfast, lunch, and dinner for your family members, with the flexibility to switch between shared meal planning and individual preferences.

## Main Features

### 📅 Week Management
- **Add/Remove Weeks** - Easily add new weeks to plan ahead or remove past weeks
- **Editable Week Dates** - Click on any week's date to customize when it starts
- **Multi-Week Planning** - Plan multiple weeks in advance

### 🍽️ Meal Planning
- **Three Daily Meals** - Organize breakfast, lunch, and dinner for each day
- **Shared Meals** - Plan meals for the whole family with shared notes
- **Individual Meals** - Switch to individual mode to track preferences for each family member
- **Dinner Details** - Break down dinners by components (Protein, Carbs, Vegetables)

### 👥 Family Member Tracking
- Track meals for multiple family members (Ian, Cristina, and Maya)
- Color-coded sections for easy visual distinction
- Flexible who's eating what for each meal

### 💾 Data Management
- **Auto-Save** - Changes are automatically saved to your browser's local storage
- **Export** - Download your meal plan as a JSON file for backup
- **Import** - Restore previously saved meal plans from JSON files
- **Data Persistence** - Your plans persist even after closing the browser

### 🎨 User-Friendly Interface
- **Responsive Design** - Works seamlessly on desktop, tablet, and mobile devices
- **Beautiful Styling** - Warm, inviting color scheme with excellent typography
- **Intuitive Controls** - Easy-to-use mode toggles and checkboxes
- **Visual Feedback** - Toast notifications confirm when data is saved

## How to Use

1. **Start Planning** - Click "+ Add Next Week" to create your first week
2. **Enter Meals** - Type in what you're planning for each meal
3. **Manage People** - Check the names of family members eating each meal
4. **Switch Modes** - Toggle between "Shared" (whole family) and "Individual" (per-person) modes
5. **Add Details** - For dinners, break down what components you're serving
6. **Save Your Work** - Everything saves automatically, or export for backup

## Technology
Built with vanilla HTML, CSS, and JavaScript. No frameworks or dependencies required. Works entirely in your browser with local storage.

## Getting Started
Simply open `index.html` in your web browser to start planning your meals!
